Ridimensionamenti rapidi per Openbox

Date ven 06 febbraio 2015 By m Tags openbox / lxde / windows manager

Openbox Ã¨ un Windows Manager per sistemi Linux utilizzato dall'ambiente desktop LXDE. Inoltre, essendo semplice nella configurazione delle associazioni di tasti (keybinding), hanno pure pensato di basarci la distribuzione Linux CrunchBang (#!), ma non siamo qui per questo!

In soldoni le associazioni di tasti di Openbox possono essere facilmente aggiunte e modificate attraverso il file rc.xml. In LXDE e in Lubuntu il file si chiama rispettivamente: lxde_rc.xml e lubuntu_rc.xml e dovrebbe trovarsi all'interno di ~/.config/openbox/

Detto questo, vediamo ora un semplice metodo per ridimensionare le finestre di Openbox!

Aprite il file rc.xml con il vostro editor di testo, e aggiungete i seguenti tags immediatamente prima del commento:

<!-- Keybindings for running applications -->

Definiamo il keybind (in questo caso tasto Windows e freccia Sinistra):

<keybind key="W-Left">

Con la prima azione richiediamo di non massimizzare la finestra:

<action name="UnmaximizeFull"/>

Ora dimezziamo la grandezza della finestra:

<action name="MoveResizeTo">
    <width>50%</width>
</action>

Massimizziamo verticalmente:

<action name="MaximizeVert"/>

Specificiamo la posizione:

<action name="MoveResizeTo">
    <x>0</x>
    <y>0</y>
</action>

Chiudiamo il tag del keybind:

</keybind>

Questo dovrebbe essere in soldoni il nostro risultato:

<keybind key="W-Left">  
  <action name="UnmaximizeFull"/>
  <action name="MoveResizeTo">
    <width>50%</width>
  </action>
  <action name="MaximizeVert"/>
  <action name="MoveResizeTo">
    <x>0</x>
    <y>0</y>
  </action>
</keybind>


Fatto! E ora?

Vi propongo un esempio di associazioni di tasti, naturalmente potete fare di meglio!

<!-- Vertical resize -->
<keybind key="W-Left">  
  <action name="UnmaximizeFull"/>
  <action name="MoveResizeTo">
    <width>50%</width>
  </action>
  <action name="MaximizeVert"/>
  <action name="MoveResizeTo">
    <x>0</x>
    <y>0</y>
  </action>
</keybind>
<keybind key="W-Right">
  <action name="UnmaximizeFull"/>
  <action name="MoveResizeTo">
    <width>50%</width>
  </action>
  <action name="MaximizeVert"/>
  <action name="MoveResizeTo">
    <x>50%</x>
    <y>0</y>
  </action>
</keybind>
<!-- Horizontal resize -->
<keybind key="W-Up">
  <action name="UnmaximizeFull"/>
  <action name="MoveResizeTo">
    <height>50%</height>
  </action>
  <action name="MaximizeHorz"/>
  <action name="MoveResizeTo">
    <x>0</x>
    <y>0</y>
  </action>
</keybind>
<keybind key="W-Down">
  <action name="UnmaximizeFull"/>
  <action name="MoveResizeTo">
    <height>50%</height>
  </action>
  <action name="MaximizeHorz"/>
  <action name="MoveResizeTo">
    <x>0</x>
    <y>50%</y>
  </action>
</keybind>

Dopo aver completato di scrivere le vostre configurazioni, salvate il file, aprite un terminale e digitate:

$ openbox --reconfigure

Avete appena chiesto ad openbox di riconfiguarsi con le nuove impostazioni. Facile no?
